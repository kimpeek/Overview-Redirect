# Overview-Redirect

### Change all username links to point to the user's overview

From:
```html
<a href="https://www.reddit.com/user/KimPeek/">KimPeek</a>
```
To:
```html
<a href="https://www.reddit.com/user/KimPeek/overview">KimPeek</a>
```
